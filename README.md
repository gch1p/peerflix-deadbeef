# peerflix-deadbeef

### Usage

```
peerflix-deadbeef <magnet link or torrent file>
```

### Requirements

Make sure that `peerflix` and `deadbeef` are in your `$PATH`.

### Install

```
git clone https://github.com/gch1p/peerflix-deadbeef
cd peerflix-deadbeef
sudo make install
```

### Uninstall

```
sudo make uninstall
```
